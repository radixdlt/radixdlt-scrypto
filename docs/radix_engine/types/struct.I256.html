Struct radix_engine::types::I256

[−]
#[repr(transparent)]
pub struct I256(pub [u8; 32]);
Expand description

Provides safe integer arithmetic.

Operations like +, ‘-’, ‘*’, or ‘/’ sometimes produce overflow which is detected and results in a panic, instead of silently wrapping around.

The bit length of output type will be the greater one in the math operation, and if any of the types was signed, then the resulting type will be signed too, otherwise the output type is unsigned.

The underlying value can be retrieved through the .0 index of the I256 tuple.

Layout

I256 will have the same methods and traits as the built-in counterpart.
