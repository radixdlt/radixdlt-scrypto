Expand description

A call frame is the basic unit that forms a transaction call stack, which keeps track of the owned objects by this function.

Fields

depth: usize

The frame id

actor: REActor

The running application actor of this frame

node_refs: HashMap<RENodeId, RENodeRefData>

All ref nodes accessible by this call frame (does not include owned nodes).

owned_root_nodes: HashMap<RENodeId, u32>

Owned nodes which by definition must live on heap Also keeps track of number of locks on this node

next_lock_handle: LockHandlelocks: HashMap<LockHandle, SubstateLock>allocated_ids: HashSet<RENodeId>
