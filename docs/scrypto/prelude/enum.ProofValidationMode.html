Expand description

Specifies the validation mode that should be used for validating a Proof.

Variants

ValidateResourceAddress(ResourceAddress)

Specifies that the Proof should be validated against a single ResourceAddress.

ValidateResourceAddressBelongsTo(BTreeSet<ResourceAddress, Global>)

Specifies that the Proof should have its resource address validated against a set of ResourceAddresses. If the Proof’s resource address belongs to the set, then its valid.

ValidateContainsNonFungible(NonFungibleAddress)

Specifies that the Proof should be validating for containing a specific NonFungibleAddress.

ValidateContainsNonFungibles(ResourceAddress, BTreeSet<NonFungibleId, Global>)

Specifies that the Proof should be validated against a single resource address and a set of NonFungibleIds to ensure that the Proof contains all of the NonFungibles in the set.

ValidateContainsAmount(ResourceAddress, Decimal)

Specifies that the Proof should be validated for the amount of resources that it contains.
